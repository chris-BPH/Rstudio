Question 1

Dans un premier temps il faut créer la variable catégorielle SCORE_SANTE_CLASSE :

sport <- sport %>%
  mutate(SCORE_SANTE_CLASSE  = cut(SCORE_SANTE, breaks = seq(0, 100, by = 10), include.lowest = TRUE))

la fonction cut crée des intervalles fermés à gauche et ouverts à droite.

include.lowest sert à inclure la valeur minimale dans la première classe de ton découpage. Même si la valeur minimal ici est 55.4.

Table vérif :

tab_verif <- sport %>% select(SCORE_SANTE_CLASSE)

Il faut faire attention dans le tableau à ce que toutes les variables soient du bon type. Ici, les variables SPORT et FREQUENCE ne sont pas du bon type : elles sont de type entier (integer) au lieu d’être un facteur (factor).

sport$SPORT <- as.factor(sport$SPORT)
sport$FREQUENCE <- as.factor(sport$FREQUENCE)
