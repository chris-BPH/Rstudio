Importation de données

Sur cette page web, je vais vous montrer comment importer des données en utilisant du code R, afin d’éviter les erreurs d’importation et de ne pas avoir à réimporter les fichiers à chaque redémarrage de R (particulièrement pratique lors des examens ou des stages).

Première étape (où est mon fichier)

Ouvre Rstudio, crée un nouveau script. Nous allons utiliser une des fonctions de R pour l’importation de données qui ne nécessite aucun package read.table().

Regarde ensuite où ton fichier à importer est placé et copie-colle son emplacement

Effectue un clic gauche puis un clic droit et clic sur “Copier en tant que chemin d’accès”.

Résultat : “C:\TUTORAT\hta.txt” tu obtiens le chemin d’accès pour que R retrouve l’emplacement de ton ficher.

Ici, l’extension .txt indique qu’il s’agit d’un fichier texte.

READ.TABLE()

Utilise la fonction read.table(), cette fonction à plusieurs paramètres pour l’importation nous allons nous concentrer sur les 4 principales.

Dans file utilise le chemin d’accès copier juste avant. ATTENTION : R ne comprend pas que / est un séparateur alors utilise //

Ainsi on passe de : “C:\TUTORAT\hta.txt”

à “C:\\Users\\\\\.txt”

et met le dans le file

Le paramètre header indique à R si la première ligne du fichier doit être considérée comme une ligne d’en-têtes, c’est-à-dire les noms des colonnes du jeu de données.

Pour dire oui/non en R on utilise TRUE/FALSE.

Le paramètre sep permet d’indiquer à R quel caractère sépare les différentes colonnes dans ton fichier de données.

Format du ficher sep = Commentaires
CSV "," Utilisé quand les colonnes sont séparées par des virgules
Excel (xls) ";" Utilisé quand les colonnes sont séparées par des points-virgules
texte "\t" ou "" Les colonnes sont séparées par une tabulation (\t) ou Les valeurs sont séparées par un espace (““)

Dans notre exemple c’est un fichier texte cependant nous ne savons pas si c’est un séparateur tabulation ou espace. Pour cela, nous devons ouvrir le fichier de données.

Passe ta souris sur le texte : si la sélection devient bleue jusqu’à la variable suivante, cela signifie qu’il y a une tabulation.

Dans le cas contraire, si la sélection s’arrête entre les mots, ce sont simplement des espaces.

Le paramètre dec permet d’indiquer à R quel symbole est utilisé pour séparer les décimales dans ton fichier de données.

Cela dépend souvent du pays d’où vient le ficher de données. Souvent les USA/UK ils utilisent . pour séparer les décimales alors que en France on utilise les ,

Le moyen le plus simple de le voir c’est de regarder le fichier de données directement.

Voilà il ne reste plus qu’a faire ctrl + entrée et ton jeu de données est importé. Vérifie quand même avec head() ou str()
